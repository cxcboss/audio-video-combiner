# 音视频合成工具

一个功能强大、界面美观的音视频合成工具，支持将音频文件与视频文件无缝合并。

## 🎯 功能特性

- **文件上传**：支持上传视频文件（MP4、AVI、MOV）和音频文件（MP3、WAV、AAC）
- **实时进度**：上传和合成过程中显示实时进度条
- **视频预览**：合成完成后可直接预览效果
- **快速下载**：一键下载合成后的视频文件
- **现代化界面**：采用渐变色彩和流畅动画的现代化设计
- **响应式布局**：适配桌面和移动设备
- **深色模式**：自动适配系统深色模式设置
- **友好交互**：平滑的悬停效果和状态通知

## 🚀 快速开始

### 方法一：直接打开

1. 下载或克隆本项目到本地
2. 直接在浏览器中打开 `index.html` 文件

### 方法二：本地服务器（推荐）

1. 下载或克隆本项目到本地
2. 在项目目录中启动本地服务器：

   ```bash
   # 使用 Python 3
   python3 -m http.server 8000
   
   # 或使用 Python 2
   python -m SimpleHTTPServer 8000
   
   # 或使用 Node.js
   npx http-server
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 📖 使用方法

1. **上传文件**
   - 点击「视频文件」区域上传视频
   - 点击「音频文件」区域上传音频
   - 系统会自动验证文件格式和大小

2. **合成视频**
   - 上传完成后，「合成视频」按钮会自动启用
   - 点击按钮开始合成过程
   - 等待合成完成（进度条会显示实时进度）

3. **预览和下载**
   - 合成完成后，会自动显示预览区域
   - 点击视频播放器可预览合成效果
   - 点击「下载合成视频」按钮保存文件

## 🛠 技术栈

- **前端框架**：纯 HTML5 + CSS3 + JavaScript
- **图标库**：Font Awesome 6
- **媒体处理**：浏览器原生 MediaRecorder API
- **样式技术**：CSS 变量、渐变、动画
- **响应式设计**：Flexbox + 媒体查询

## ⚙️ 技术实现

### 文件验证
- 检查文件大小（视频最大 200MB，音频最大 50MB）
- 验证文件格式（支持常见音视频格式）

### 视频合成
- 使用浏览器原生 `MediaRecorder API` 实现音视频合成
- 支持多种输出格式（WebM、MP4 等）
- 保持原始视频质量

### 界面交互
- 实时状态通知
- 平滑的过渡动画
- 响应式设计适配
